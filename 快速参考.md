## 📋 三大核心功能

### 1️⃣ 染色配方计算
**用途**: 根据目标颜色和环境条件，自动计算染料配比

**快速调用**:
```python
from color_recognizer import DyeFormulaCalculator

calc = DyeFormulaCalculator()
result = calc.calculate_dye_formula(
    target_color_rgb=[220, 20, 60],  # RGB颜色
    ambient_temp=25,                  # 温度℃
    ambient_humidity=60               # 湿度%
)

print(result['dye_ratio'])   # 染料比例
print(result['soak_time'])   # 浸泡时间（分钟）
print(result['fix_time'])    # 固色时长（分钟）
```

**支持染料**: 红花、板蓝根、黄刺叶、紫草、茶叶、核桃皮、洋葱皮、石榴皮

---

### 2️⃣ 品质检测
**用途**: 自动检测绸缎成品的染色均匀度、纹样对齐度和瑕疵

**快速调用**:
```python
from quality_inspection import QualityInspectionApp
import cv2
import tkinter as tk

root = tk.Tk()
app = QualityInspectionApp(root)
image = cv2.imread("fabric.jpg")

results = app.inspect_quality(image)
print(f"等级: {results['grade']}")
print(f"均匀度: {results['dye_uniformity']}%")
print(f"对齐度: {results['alignment_degree']}%")
print(f"瑕疵数: {results['defect_count']}")
```

**分级标准**:
- 精品级: 均匀度≥95%, 对齐度≥98%, 瑕疵率≤0.1/万
- 合格级: 均匀度≥90%, 对齐度≥95%, 瑕疵率≤0.3/万
- 待返工: 其他情况

---

### 3️⃣ 纹样生成
**用途**: AI智能生成符合传统工艺的艾德莱斯绸纹样

**快速调用**:
```python
from pattern_design_u1 import SilkPatternDesignApp
import tkinter as tk

root = tk.Tk()
app = SilkPatternDesignApp(root)

user_pref = {
    'style_type': '国潮现代',
    'color': '中国红',
    'density': 'medium',
    'symmetry': 0.85
}

pattern = app.generate_edelis_pattern(
    base_style='traditional',
    user_preference=user_pref,
    pattern_database=app.PATTERN_DATABASE
)

print(f"线条宽度: {pattern['line_width']}px")
print(f"配色方案: {pattern['colors']}")
print(f"织造就绪: {pattern['weaving_ready']}")
```

**支持风格**: 传统经典、国潮现代、旅拍简约  
**经典配色**: 中国红、帝王黄、宝石蓝、翡翠绿等12色

## 📊 关键参数速查

### 染色配方
| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| target_color_rgb | list | 目标颜色RGB | 必填 |
| ambient_temp | float | 环境温度℃ | 25 |
| ambient_humidity | float | 环境湿度% | 60 |

### 品质检测
| 指标 | 范围 | 说明 |
|------|------|------|
| dye_uniformity | 0-100% | 染色均匀度 |
| alignment_degree | 0-100% | 纹样对齐度 |
| defect_rate | 浮点数 | 每万像素瑕疵数 |

### 纹样生成
| 参数 | 类型 | 可选值 |
|------|------|--------|
| style_type | str | traditional/国潮现代/旅拍简约 |
| color | str | 12种经典色之一 |
| density | str | low/medium/high |
| symmetry | float | 0-1 |
