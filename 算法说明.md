## 1. 染色配方智能计算算法

### 1.1 算法概述
基于**色彩回归+环境因子修正**的智能染色配方计算系统，实现从目标颜色到天然染料配方的精准转换。

### 1.2 核心功能

#### ✨ RGB转LAB色彩空间
- **目的**: 提升色彩匹配精度
- **方法**: 使用CIE标准转换公式
- **优势**: LAB色彩空间更符合人眼感知

```python
target_lab = rgb_to_lab(target_color_rgb)
```

#### 🔍 天然染料数据库匹配
- **支持染料**: 红花、板蓝根、黄刺叶、紫草、茶叶、核桃皮等本土染料
- **匹配算法**: CIEDE2000色差计算（简化版）
- **阈值控制**: 默认色差阈值3%

```python
dye_candidates = query_dye_database(dye_database, target_lab, error_threshold=3)
```

#### 📊 线性回归模型
- **功能**: 拟合染料比例与色彩的关系
- **方法**: 基于相似度加权平均
- **输出**: 各染料的精确配比

```python
base_formula = linear_regression_model(target_lab, dye_candidates)
```

#### 🌡️ 环境因子修正
- **温度修正**: `temp_correction = 1 + (ambient_temp - 25) * 0.015`
- **湿度修正**: `humidity_correction = 1 + (60 - ambient_humidity) * 0.012`
- **标准条件**: 温度25℃，湿度60%

```python
final_formula = {
    dye: ratio * temp_correction * humidity_correction 
    for dye, ratio in base_formula.items()
}
```

### 1.3 输出参数
- **dye_ratio**: 染料比例字典
- **soak_time**: 浸泡时间（分钟）
- **fix_time**: 固色时长（分钟）
- **environmental_conditions**: 环境修正因子详情

### 1.4 使用示例
```python
from color_recognizer import DyeFormulaCalculator

calculator = DyeFormulaCalculator()
result = calculator.calculate_dye_formula(
    target_color_rgb=[220, 20, 60],  # 中国红
    ambient_temp=28,                  # 环境温度28℃
    ambient_humidity=55               # 环境湿度55%
)

print(f"染料比例: {result['dye_ratio']}")
print(f"浸泡时间: {result['soak_time']}分钟")
print(f"固色时长: {result['fix_time']}分钟")
```

## 2. 品质检测算法

### 2.1 算法概述
基于**YOLOv8+图像分割**的成品质量检测系统，实现染色均匀度、纹样对齐度、瑕疵检测三大核心指标评估。

### 2.2 核心功能

#### 🖼️ 图像预处理
- **去噪**: fastNlMeansDenoisingColored算法
- **对比度增强**: CLAHE自适应直方图均衡化
- **透视校正**: 基于边缘检测和轮廓提取

```python
processed_img = image_preprocess(image, operations=[
    "denoise", 
    "contrast_enhance", 
    "perspective_correct"
])
```

#### 📈 染色均匀度检测
- **方法**: 颜色方差计算
- **公式**: `uniformity = 100 - (variance / max_variance) * 100`
- **评价标准**: ≥95%为精品级

```python
color_variance = calculate_color_variance(processed_img)
dye_uniformity = 100 - (color_variance / standard_params["max_variance"]) * 100
```

#### 🎯 纹样对齐度检测
- **特征点提取**: ORB检测器
- **角度分析**: 计算特征点连线角度分布
- **对齐度计算**: 基于角度标准差

```python
pattern_keypoints = detect_pattern_keypoints(processed_img)
alignment_degree = calculate_alignment(pattern_keypoints)
```

#### 🔎 瑕疵检测（3类核心瑕疵）
1. **破洞 (hole)**: 暗区域检测
2. **污渍 (stain)**: 颜色异常区域
3. **线头错误 (thread_error)**: 边缘异常检测

```python
defects = yolov8_detect(processed_img, defect_classes=[
    "hole", 
    "stain", 
    "thread_error"
])
defect_rate = len(defects) / (image_pixels / 10000)  # 每万像素瑕疵数
```

### 2.3 产品分级标准

| 等级 | 染色均匀度 | 纹样对齐度 | 瑕疵率 | 说明 |
|------|-----------|-----------|--------|------|
| 精品级 | ≥95% | ≥98% | ≤0.1/万 | 出口品质 |
| 合格级 | ≥90% | ≥95% | ≤0.3/万 | 内销品质 |
| 待返工 | <90% | <95% | >0.3/万 | 需要修复 |

### 2.4 输出参数
- **dye_uniformity**: 染色均匀度（%）
- **alignment_degree**: 纹样对齐度（%）
- **defect_rate**: 瑕疵率（每万像素）
- **grade**: 产品等级
- **defect_positions**: 瑕疵坐标列表（用于返工定位）

### 2.5 使用示例
```python
from quality_inspection import QualityInspectionApp
import cv2

app = QualityInspectionApp(root)
image = cv2.imread("fabric.jpg")
results = app.inspect_quality(image)

print(f"等级: {results['grade']}")
print(f"均匀度: {results['dye_uniformity']}%")
print(f"对齐度: {results['alignment_degree']}%")
print(f"瑕疵数: {results['defect_count']}")
```

## 3. 纹样生成算法

### 3.1 算法概述
基于**StyleGAN3+传统特征约束**的智能纹样生成系统，融合用户偏好与传统工艺要求，生成可织造的艾德莱斯绸纹样。

### 3.2 核心功能

#### 🎨 传统纹样特征提取
- **花瓣曲线 (petal_curve)**: 曲线平滑度
- **线条密度 (line_density)**: 纹样密集程度
- **色彩和谐度 (color_harmony)**: 配色协调性

```python
pattern_features = extract_traditional_features(
    pattern_database, 
    feature_keys=["petal_curve", "line_density", "color_harmony"]
)
```

#### 🧬 风格编码器
- **输入**: 基础风格 + 用户偏好
- **输出**: 风格嵌入向量
- **支持风格**: 传统经典、国潮现代、旅拍简约

```python
style_embedding = style_encoder(base_style, user_preference["style_type"])
```

#### 🌈 艾德莱斯配色方案
- **经典12色**: 中国红、帝王黄、宝石蓝、翡翠绿等
- **智能适配**: 根据色彩和谐度调整配色

```python
color_palette = get_edelis_color(
    user_preference["color"], 
    pattern_features["color_harmony"]
)
```

#### ⚙️ 生产工艺约束
- **线条粗细**: 2-5px（适配扎经染色工艺）
- **图案重复**: True（满足织造循环性）
- **对称度控制**: 0-1可调

```python
generation_constraint = {
    "line_stroke": (2, 5),
    "pattern_repeat": True,
    "symmetry_degree": user_preference["symmetry"]
}
```

#### 🔄 织造优化
- **线条宽度调整**: 确保在2-5px范围内
- **重复性保证**: 图案可无缝拼接
- **工艺适配**: 符合手工织造要求

```python
raw_pattern = stylegan3_generate(style_embedding, color_palette, constraints)
optimized_pattern = adjust_for_weaving(raw_pattern)
```

### 3.3 输出参数
- **style**: 风格嵌入向量
- **colors**: 配色方案（4色）
- **line_width**: 线条宽度（px）
- **repeat_pattern**: 是否重复
- **symmetry**: 对称度
- **complexity**: 复杂度（0-1）
- **weaving_ready**: 是否织造就绪

### 3.4 使用示例
```python
from pattern_design_u1 import SilkPatternDesignApp

app = SilkPatternDesignApp(root)
user_preference = {
    "style_type": "国潮现代",
    "color": "中国红",
    "density": "medium",
    "symmetry": 0.85
}

pattern = app.generate_edelis_pattern(
    base_style="traditional",
    user_preference=user_preference,
    pattern_database=app.PATTERN_DATABASE
)

print(f"线条宽度: {pattern['line_width']}px")
print(f"配色: {pattern['colors']}")
print(f"织造就绪: {pattern['weaving_ready']}")
```

## 常见问题

### Q1: 为什么蓝色系染料匹配失败？
A: 当前染料数据库中蓝色染料较少（仅板蓝根），建议扩充数据库或放宽色差阈值。

### Q2: 如何提高瑕疵检测精度？
A: 集成真实的YOLOv8模型，并使用标注好的瑕疵数据集进行微调。

### Q3: 生成的纹样可以直接织造吗？
A: 算法已考虑织造约束，但实际生产前仍需工匠审核调整。

### Q4: 如何添加自定义纹样模板？
A: 在 `patterns/` 目录下按格式添加图片文件，命名规则：`分类_序号.jpg`
